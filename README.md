# ArduCleaner - Arduino Sweeper robots

## Required components

- Arduino Mega 1280/2560
- HC-SR04 ultrasonic sensor
- L298 2Amp Motor Driver Shield
- Two wheels and two DC motors
- 3D Printer (for the case)
